Install:

```
$ composer install
```

Run:

```
$ php5 -S localhost:8000
(OR Apache)
``` 

Test:

```
http://[IP]:8000/hello/xxxxxxxx

http://[IP]:8000/hello2/xxxxxxxx
```
if the index.php file name was other like api.php then use:
```
http://[IP]:8000/filename.php/hello/xxxxxxxx
```
